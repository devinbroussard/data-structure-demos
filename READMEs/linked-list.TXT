1. Goal of application:
	This application was made to demonstrate an implementation of doubly linked list.
2. Demonstration steps:
	1. Adding items to the list
	2. Copying items from one list onto another
	3. Sorting items in a list using insertion sort
	4. Removing an item from a list
	5. Inserting an item in a list
3. How to play:
To open the application, download the zip file, extract, and run the ".exe" file.
